Leading Health Care Provider

Portal is being developed for leading health care provider of US. Portal aims to simplify the existing process of client with help of BPM. Portal will allow users to enroll into the system and request for the services of client using web while BPM will manage all the process in the backend enabling client to fulfill user requests in an efficient way. Portal allows surgeons to submit the reports of their patients on the web, based on reports client determines the technical details of the body part to be manufactured. At the end of the process client manufactures the body part of the patient and ship it back to the facility/healthcare system for the surgery.

Lottery System

Portal also allows anonymous user to track recent winning numbers and winning amount along with the information of the next due date of draw and proposed jackpot amount for different games from the public facing site developed using port lets and WCM. User can also claim his winning amount with the help of the portal.
